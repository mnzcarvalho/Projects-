===player===

level 1 ~ 100
3 points per lvl
change class lvl 30
master classe change lvl 70
class change = warrior(str), mage(int), archer(dex)

skill tree = each class have 1 active skill and passive skills in skill tree (beginner class not included)

===status===

strength - base 5 - physical damage

agility - base 5 -  vel atk (max 2 attacks per second) (-0.01 atk per second per points)

intelligence - base 5 - spell damage

vitality - base 5 - increased HP - 1 hp per point


===equipment===

set - helmet, armor, gloves, pants e boots

weapons -1h sword (beginner), 2h axe(Warrior), Staff(Mage), Bow(Archers) 

accessories - left ring, right ring, necklace
